Инструкция:

Открываем tgpars.py через Nodepad++

Ставим свои значения из телеги:

api_id = 'ВАШ API ID'
api_hash = 'ВАШ API HASH'
phone_number = 'Номер от телеги аккаунт'
password = 'Код пароль от вашей телеги'

Как получить API ID и API HASH:
Заходим по ссылке https://my.telegram.org/auth?to=apps  
(Обратите внимание при переходе по ссылке будет такой текст Удалить учетную запись или управлять приложениями)

Просто вводим номер телефона с вашим аккаунтом ничего не случится.
После получаете код приложения от Телеграм в своем приложении.
Вводите его для авторизации
Отлично вы увидите свой API ID и API HASH
Название приложения: тут пишите на рандом
Короткое имя: тут на рандом
Сохраняем.

в groups.txt указываем группы с открытыми участниками для парса.

После того как вы успешно заменили все эти пункты на свои значения. 

запускаем tgpars.py запускает через cmd а не просто кликом по нему.
Парсер запросит код который прийдет вам в телегу вводите его и софт начинает свою работу.
Не забудьте установить нужную библиотеку pip install telethon


**
После того как вы запустите софт вам подгрузит сессию не удаляйте её чтобы не авторизироватся каждый раз через телегу.
